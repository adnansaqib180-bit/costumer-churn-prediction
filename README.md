# 📉 Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to churn or stay. The model is built using Scikit-Learn Pipelines and optimized with GridSearchCV, then deployed with Streamlit for real-time predictions.

## 🚀 Live Demo

https://costumer-churn-prediction-by-adnan.streamlit.app/

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* Streamlit
* Joblib

## 🤖 Models Used

* Logistic Regression
* KNN
* SVM

## ⚙️ Features

* Data Preprocessing Pipeline
* GridSearchCV Hyperparameter Tuning
* Real-Time Predictions
* Interactive Streamlit Interface
